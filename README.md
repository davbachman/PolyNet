# PolyNet

An interactive 3D polyhedron visualization tool. Select a Platonic solid, apply geometric transformations, unfold it into a flat net, and export the result.

**[Launch PolyNet](https://davbachman.github.io/PolyNet/)**

## How to Use

1. **Choose a solid** -- Use the dropdown in the toolbar to pick from the five Platonic solids: tetrahedron, cube, octahedron, dodecahedron, or icosahedron.

2. **Transform it** -- Click **+ Stellate** or **+ Truncate** in the sidebar to add operations. Drag the slider to control the amount. Stack multiple operations to create complex shapes. Remove any step with the **x** button.

3. **Unfold it** -- Click the **Unfold** button to animate the solid unfolding into a flat net. Click **Fold** to reverse the animation.

4. **Rotate the view** -- Click and drag on the 3D canvas to orbit around the solid. Scroll to zoom in and out.

5. **Export** -- Use **Export OBJ** to download a 3D model file, or **Export PNG** to save an image of the unfolded net.
